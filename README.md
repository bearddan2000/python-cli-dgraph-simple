# python-cli-dgraph-simple

## Description
POC for dgraph in python.

## Tech stack
- python
  - pydgraph
  - grpcio

## Docker stack
- python:latest
- dgraph/dgraph:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
